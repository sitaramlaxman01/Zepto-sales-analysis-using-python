# Zepto-sales-analysis-using-python
🚀 Data Analysis Project using Python

I recently completed a data analysis project on Zepto customer data to understand user behavior and business insights.

Key Insights:
• Most orders are placed by male customers
• Age group 46–60 contributes the highest orders
• Groceries and automobiles are top-selling categories
• Positive sentiment dominates customer reviews
• Product preferences vary by gender
Tools Used:
  Python, Pandas, Data Visualization

This project helped me understand how data can be used to make business decisions and improve customer experience.
